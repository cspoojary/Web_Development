# ID Card UI Design – HTML & CSS

## Project Overview
This project is a **custom-designed ID Card UI** developed using **HTML5 and CSS3**.  
It demonstrates structured layout design, visual styling, and clean UI composition.

The project was built **during my learning process** to apply web development concepts in a practical, real-world style interface.

---
## Technologies Used
- HTML5
- CSS3
- Flexbox Layout

## Features
- Center-aligned ID card layout

- Clean typography and color contrast
- Organized personal and professional details
- Responsive-friendly card structure
